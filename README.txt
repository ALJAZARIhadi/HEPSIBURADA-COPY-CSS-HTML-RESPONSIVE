Proje Analiz ve Bilgi Dosyası (README)

1. Proje Hakkında
Bu proje, popüler e-ticaret platformu olan Hepsiburada web sitesinin ana sayfa yerleşim yapısını (layout) HTML ve CSS kullanarak yeniden oluşturma ödevidir. Projenin odak noktası, görsel tasarımın birebir kopyalanmasından ziyade, sayfanın iskeletini ve kolon (grid) yapısını doğru kurmaktır.
+2

2. Seçilen Site Bilgileri


Site Adı: Hepsiburada 

Site Linki: https://www.hepsiburada.com/

3. Kolon ve Yapı Analizi
Seçilen sitenin ana sayfa yapısı incelenmiş ve aşağıdaki kolon planı uygulanmıştır:


Header: Logo, arama çubuğu ve kullanıcı aksiyonlarını içeren üst alan.


Navigation: Kategorilerin listelendiği yatay menü çubuğu.


Main Alanı (İki Kolonlu Yapı): 


Sol Kolon (Sidebar): Filtreleme seçenekleri ve kategori listesini içeren yan menü.


Sağ Kolon (Ana İçerik): Bannerlar ve ürün kartlarının bulunduğu alan.


Ürün Kartları: Grid yapısı kullanılarak 6 adet içerik kartı yan yana dizilmiştir.


Footer: Kurumsal bilgiler ve sosyal medya linklerini içeren alt bölüm.

4. Responsive (Mobil Uyumluluk) Planı
Tasarım, farklı ekran boyutlarına göre şu şekilde uyarlanmıştır:


Desktop (≥1024px): Sidebar ve ana içerik yan yana, ürünler 3'lü grid yapısında.


Tablet (768px - 1023px): Ürün kartları 2'li grid yapısına düşürülmüş, yerleşim korunmuştur.


Mobil (≤767px): Tüm yapı tek kolona düşürülmüş, içerikler (Sidebar ve Kartlar) alt alta gelecek şekilde düzenlenmiştir.

5. Kullanılan Teknolojiler

Sadece Semantik HTML5 etiketleri kullanılmıştır.
+1

Düzenleme için CSS Flexbox ve CSS Grid yapıları tercih edilmiştir.

Harici bir kütüphane (Bootstrap vb.) kullanılmamıştır.
